<h3>Application Setup:</h3>
<ul>
    <li>$ git clone https://github.com/AlexPurhalo/sinatra-blog.git</li>
    <li>$ cd sinatra-blog</li>
    <li>$ rvm install 1.9.3</li>

</ul>
